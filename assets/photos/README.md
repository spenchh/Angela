# Photos for the "little moments" slide

Drop Angela's photos here, named exactly:

photo-1.jpg
photo-2.jpg
photo-3.jpg
...
photo-10.jpg

(.jpg, .jpeg, .png, .webp all work — but the paths in index.html
`personalPhotos` expect `.jpg`, so either rename your files to `.jpg`
or update the extensions in that list.)

Any slot whose file is missing will gracefully show a watercolor
ink-wash placeholder instead, so it's fine to add them a few at a time.
